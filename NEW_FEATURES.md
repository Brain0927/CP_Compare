✅ **提取商品資訊**

- 評論

✅ **分析商品類別**

### 3️⃣ **完整對比流程**

#### 價格比較
在「🚀 完整流程」中：
- 自動顯示每個商品的價格
- 按價格從低到高排列
- 清晰看出價差

**範例表格**:
```
| 商品 | 價格 | CP值 | 評分 |
|------|------|------|------|
| 機型A | $25,000 | 0.95 | 4.5⭐ |
| 機型B | $30,000 | 0.88 | 4.3⭐ |
| 機型C | $35,900 | 0.78 | 4.8⭐ |
```

#### CP值比較
```
公式: CP = Σ(特徵分數 × 權重) / 價格 × 評分加成

範例:
- 機型A: 0.95 CP值 (特徵好 + 便宜)
- 機型C: 0.78 CP值 (特徵好 + 貴) ← 評分雖然最高，但價格高
```

#### AI 推薦
系統自動生成中文推薦理由：
- 為什麼推薦這個商品？
- 各商品的優勢
- 適合的使用場景
- 綜合評價

**推薦範例**:
```
🏆 第1名推薦: 機型A
- 性價比最高 (CP值 0.95)
- 價格便宜 ($25,000)
- 評分優良 (4.5⭐)
- 適合: 日常使用、預算考慮

🥈 第2名推薦: 機型B
- 效能均衡 (CP值 0.88)
- 價格中等 ($30,000)
- 適合: 專業工作、效能需求
```

## 🎓 完整使用教程

### 場景: 購買一台新筆電

#### 第1步: 單品分析
1. 打開應用
2. 切換到「🔍 單品分析」
3. 貼入心儀筆電的連結
   ```
   https://www.momoshop.com.tw/product/MacBook-Pro
   ```

5. 查看分析結果和搜尋建議

**得到的信息**:

#### 第2步: 尋找相似商品
根據搜尋建議，尋找 2-3 個類似的筆電：
- Dell XPS 13
- ASUS VivoBook 14
- Lenovo ThinkPad

#### 第3步: 完整對比
1. 切換到「🚀 完整流程」

3. 點擊「開始爬取」

5. 調整權重 (例如: CPU 3分, 續航 2分)
6. 查看結果

**最終得到**:
```
CP值排行:
1️⃣ MacBook Pro (CP值: 0.85)
2️⃣ Dell XPS (CP值: 0.92) ← 最佳性價比
3️⃣ ASUS VivoBook (CP值: 1.05) ← 超優 CP 值
```

#### 第4步: 做出決定
根據 AI 推薦和 CP 值分析，選擇最適合的筆電。

## ⚠️ 常見問題

### Q: 如何開始？
A: 進入「🔍 單品分析」，貼上一個商品連結就能開始。

### Q: 能否分析所有網站的商品？
A: 支援大多數 HTML 頁面。Momo、PChome、蝦皮最佳支援。

### Q: 為什麼有時提取失敗？
A: 可能原因：
- URL 無效或頁面被下架
- 網站有反爬蟲保護
- 頁面結構不同尋常
解決: 試試其他商品或選擇「動態載入」模式。

### Q: CP值低的商品不能買？
A: 不一定。CP值只是一個參考指標，應考慮：
- 個人需求和偏好
- 品牌信譽和保修
- 評論質量和反饋
