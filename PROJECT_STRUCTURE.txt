🎯 使用流程
═══════════════════════════════════════════════════════════════════════════

1️⃣  輸入商品連結
    └─ 支援 2-4 個商品，選擇爬蟲模式

2️⃣  自動爬取
    └─ 擷取名稱、價格、規格、評論

3️⃣  NLP 分析
    └─ 使用 Gemini API 分析特徵重要性

4️⃣  權重調整
    └─ 手動微調 1-3 分的特徵權重

5️⃣  結果展示
    └─ CP 值排行、表格、圖表、推薦理由


📐 CP 值公式
═══════════════════════════════════════════════════════════════════════════

CP = Σ(Feature × Weight) / Price × (1 + Rating/5 × 0.2)

其中:
  - Feature: 特徵歸一化分數 (0-1)
  - Weight: 特徵權重 (1-3)
  - Price: 商品價格
  - Rating: 用戶評分加成 (最多 +20%)


🚀 快速啟動
═══════════════════════════════════════════════════════════════════════════

# 1. 建立虛擬環境
python3 -m venv venv
source venv/bin/activate

# 2. 安裝依賴
pip install -r requirements.txt

# 3. 配置 API
cp .env.example .env
# 編輯 .env，填入 GEMINI_API_KEY

# 4. 快速測試
python test.py

# 5. 啟動應用
streamlit run app.py
